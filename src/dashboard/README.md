# Streamlit Dashboard UI

This folder contains the visualization frontend built with Streamlit.

## Features

1. **Market Overview**: Live charts of Sensex & Bank Nifty candles.
2. **AI Predictions**: Direction probability gauges, multi-horizon ranges, and confidence metrics.
3. **News Intelligence Feed**: Real-time listing of incoming news events and their retrieved historical event counterparts.
4. **LLM Explainer**: Renders natural language summaries explaining predictions.
5. **Trade Logs**: Lists paper-trading performance, active holdings, and profit-and-loss statements.

## Running the Dashboard

```bash
streamlit run src/dashboard/app.py
```
